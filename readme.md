## Descripción

## Modelo de datos
Se implementa 4 base de datos que almacenen datos del cliente, pedidos, productos y mesa. EL de cliente almacena los datos del cliente, el del producto tiene el menu y su lista de precios, el de mesa indica el estado de las mesas y cada uno se relaciona con el pedido, donde se encarga de chequear los pedidos.
